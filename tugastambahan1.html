Contoh Tag <a> dan Jenis Hyperlink

Halaman ini mendemonstrasikan berbagai tipe hyperlink: absolute, relative, email, dan targeting (atribut target).

Absolute Relative Email Targeting

Absolute Hyperlink

URL lengkap dengan protokol, domain, dan path.

Contoh: Kunjungi contoh produk (example.com) 

<a href="https://www.example.com/products/item-123">...</a>

Relative Hyperlink

URL relatif terhadap lokasi file saat ini.

Relatif ke folder yang sama:

<a href="tentang.html">Tentang Kami</a>

(Demo) tentang.html

Relatif ke subfolder:

<a href="/assets/docs/panduan.pdf">Panduan</a>

(Demo) /assets/docs/panduan.pdf

Relatif naik satu folder:

<a href="../index.html">Kembali ke Beranda</a>

(Demo) ../index.html

Email Hyperlink

Membuka aplikasi email dengan alamat/subject terisi.

Email Support 

<a href="mailto:support@contoh.co.id?subject=Butuh%20Bantuan&body=Halo">Email Support</a>

Targeting Hyperlink (atribut target)

Mengontrol di mana link dibuka.

  • Buka di tab/rame yang sama (_self)
  • Buka di tab baru (_blank)
  • Loncat ke bagian Absolute (fragment di halaman yang sama)
  • Buka di nama jendela/frame khusus (contentFrame) 
<a href="https://..." target="_blank" rel="noopener noreferrer">Buka tab baru</a>

Catatan: beberapa tautan demo mungkin 404 jika file tidak ada di proyek Anda. Kode ditujukan sebagai contoh penulisan.
